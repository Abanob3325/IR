Responsible Crawling

04

08 · Quality

Data Quality Management

08

Missing Values

Budget fields missing are imputed using mean imputation. Non-numeric descriptions are stored as null.

Deduplication

A seen_urls set prevents redundant requests during scraping, and Pandas drop_duplicates ensures uniqueness by URL in preprocessing.

Fault Tolerance

Every parser is wrapped in try/except. A malformed HTML card logs a warning and is skipped — the loop never crashes.

Quality Principle: The pipeline follows a "graceful degradation" design — every field has a fallback of null. The output is always a valid JSON file.

11 · Core IR Concepts

Information Retrieval Engine

11

Implemented Data Structures

Inverted Index: Maps each term to its Document Frequency and a Posting List of doc IDs.
Positional Index: Stores exact token offsets enabling Exact Phrase Searching.
TDIM (Term-Document Matrix): Binary matrix tracking presence (1) or absence (0) of terms.

Ranking & Semantics

TF-IDF Vector Space Model (VSM)

Documents and queries are mapped into high-dimensional space. We compute TF-IDF scores to rank search results by relevance rather than mere keyword presence.

Latent Semantic Analysis (LSA)

Applies SVD to reduce the TF-IDF matrix into semantic components, capturing conceptual similarities (e.g., "React" and "Frontend").
